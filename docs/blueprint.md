# **App Name**: Idea Stream

## Core Features:

- Idea Submission: Allow users to submit their ideas via a simple form.
- Idea Display: Display submitted ideas in a visually appealing and prominent manner.
- Data Storage: Store and retrieve ideas in a structured format to maintain data persistence.
- Real-time Updates: Display all ideas instantly as they are submitted without needing to refresh the page.
- Creativity Ranker: Use Gemini AI to analyze new ideas and measure their originality, then sort ideas by creativity score.

## Style Guidelines:

- Primary color: Light orange (#FFB347) to inspire creativity.
- Background color: Very light orange (#FFE0B2) to keep the design simple.
- Accent color: Orange (#FF9800) for idea submission button.
- Body and headline font: 'PT Sans' for a clean and accessible style.
- Clean layout for easy submission and quick browsing.
- Minimal use of icons to keep the interface uncluttered.
- Smooth transitions when ideas are displayed.
